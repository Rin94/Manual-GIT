Manual de GIT

Comandos e instrucciones

Inicializar repo
git init
Agregar elemento al stagging

git add nombre

git add * Para todos los elementos de un directorio

Ver el flujo de como estan nuestros documentos

git status

git status -s Nos muestra los que estan modificados y faltan

Ver el log historico de commits

git log

git log --oneline No los muestra en una linea

Ir a un commit anterior, restaurando el archivo

git reset --hard (codigoInstantanea)

Agregar un elemento al stagging area y hacer el commit

git commit -am "texto"

Modificar la descripcion de un commit

git commit --amend

Subir a GITHUB/GITLAB nuestros trabajos

Subir a github

git remote add origin https://github.com/Rin94/Manual-GIT.git

Actualizar a git nuestros trabajos

git push -u origin master(rama)

Traer cambios desde github o remoto

git pull

Decir la version de nuestro proyecto

Mediante TAGs

git tag textoVersion -m "Descripcion del proyecto"

Despues subirlos

git push --tags

Clonar desde un repo externo

git clone https://github.com/Rin94/Manual-GIT.git

nuestros cambios desde nuestro último commit y almacenarlos de forma temporal dejando nuestro diectorio de trabajo completamente limpio

git stash

Regresar a nuestro trabajo despues de un stash

git stash pop
